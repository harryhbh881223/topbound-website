導電矽膠條與電熱產品

以導電矽膠與絕緣矽膠層層堆疊壓合而成，用於 LCD 顯示器與電路板間的訊號導通連接，是熱壓排線與傳統連接器的可靠替代方案。

導電矽膠條

T / T2 TYPE

T（T2）type

使用導電矽橡膠和絕緣矽橡膠一層一層相互堆疊而成，T、T2 type 的設計原理為有一至數個導電體接觸 PCB 或 LCD 的電極表面。

通常應用於電子錶、電子鐘、溫度計、汽車及工業儀表、LCD 模組等。

* T2 type：為兩側邊噴塗白色絕緣體。

TS TYPE

TS type

利用 T type 的材質作中間層支撐，在兩側披覆上比 T type 硬度較低的絕緣矽橡膠，較適宜用在較大的 LCD 顯示器上使用。

通常使用在 LCD 顯示電子鐘錶、汽車儀表、工業測試儀表、掌上型電玩、遙控器、LCD 液晶顯示器模組等。

TG TYPE

TG type

利用 T type 的材質作中間層支撐，在兩側加上發泡海綿狀的絕緣矽橡膠，硬度比標準型 TS type 側邊絕緣橡膠更軟，因此也適用於大型 LCD 顯示器，且表面光滑、更易於裝配。

適用於 LCD 顯示的電子鐘錶、遙控器、工業儀表、音響、收音機、LCD 液晶顯示模組等。
